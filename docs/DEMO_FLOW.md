# Demo Flow (Presentation Script)

Project made by Shagun Talwar

1. Open landing page and explain the project objective: AI-assisted brain MRI analysis prototype.
2. Highlight medical disclaimer and no-staging policy.
3. Go to Upload page and upload first MRI with patient details.
4. Show analysis output cards: tumor detection, type, confidence, risk, runtime mode.
5. Show MRI, mask, Grad-CAM, and overlay previews.
6. Download/open generated PDF report.
7. Upload a second MRI for the same patient (different scan date).
8. Open Patient History and show timeline cards and progression chart.
9. Open Comparison page, compare old/new scan IDs, and explain change metrics and progression index.
10. Open Model Metrics page and explain real vs demo status.
11. Conclude with limitations, expert-review requirement, and future scope.
