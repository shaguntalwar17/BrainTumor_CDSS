# Demo Flow (Presentation Script)

Project made by Shagun Talwar

1. Open landing page and explain the project objective: AI-assisted brain MRI analysis prototype.
2. Show the left sidebar navigation (Home, Upload MRI Image, Patient History, Compare Reports, Awareness, Report Download).
3. Highlight medical disclaimer and research-stage limitation policy.
4. Go to Upload page and upload first MRI with patient details.
5. Show automatic patient ID generation / matching strategy feedback.
6. Show analysis output cards: detection, type, stage estimate, confidence, risk, runtime mode.
7. Show uncertainty score and explainability consistency with warning behavior.
8. If NIfTI is uploaded, demonstrate interactive 3D slice viewer scrolling.
9. Show MRI, mask, Grad-CAM/HiResCAM-style map, and overlay previews.
10. Upload a human-corrected mask from Dashboard to demonstrate active-learning feedback loop.
11. Download/open generated PDF report.
12. Upload a second MRI for the same patient (different scan date).
13. Open Patient History and show timeline cards, corrected mask visibility, and progression chart.
14. Use the Smart RAG Summary panel on History page and ask: "How has tumor size changed over time?"
15. Open Comparison page, compare old/new scan IDs, and explain change metrics, progression index, growth/reduction map, and stage change.
16. Open Report Download page and generate a comparison report PDF.
17. Open Awareness page for educational content.
18. Open Model Metrics page and explain real vs demo status.
19. Conclude with limitations, expert-review requirement, and future scope.
