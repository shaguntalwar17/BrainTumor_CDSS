# Viva / Demo Script

Project made by Shagun Talwar and co-developed by Chirag Garg

## 1. Opening (30-45 sec)
- Introduce system as an AI-assisted research prototype for brain MRI analysis.
- Mention explicit non-clinical disclaimer.

## 2. Landing Page
- Show feature scope: detection, segmentation, classification, Grad-CAM, longitudinal comparison, RAG, reports.

## 3. Upload Demo
- Enter patient details and upload sample MRI.
- Explain preprocessing, inference, uncertainty warning, and class probabilities.
- Show MRI/mask/Grad-CAM/overlay preview panels.

## 4. Report Demo
- Open/download generated PDF.
- Walk through sections and limitation statements.

## 5. Patient History
- Search same patient ID.
- Show timeline cards and progression chart.
- Open older report from history.

## 6. Longitudinal Comparison
- Compare two scan IDs.
- Explain absolute/percentage change and progression index.
- Show side-by-side previous/current assets.

## 7. Model Performance
- Open model leaderboard page.
- Explain sample vs actual metrics policy.
- Discuss why F1/recall/AUC matter in medical settings.

## 8. RAG Showcase
- Ask a patient-history query.
- Highlight grounded response and citations by scan date/id.
- Mention that responses are generated only from stored patient records (no external diagnosis text).

## 9. Closing
- Reiterate novelty:
1. Longitudinal progression index
2. Explainability consistency scoring
3. RAG-grounded historical summaries
- Reiterate disclaimer and future scope.
